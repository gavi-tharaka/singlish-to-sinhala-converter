# Singlish to Sinhala Unicode Converter

A lightweight, zero-dependency, global JavaScript utility that automatically converts Singlish phonetics into real-time Sinhala Unicode inside any web page's input boxes and textareas.

## Features
- **Zero Dependencies**: Pure Vanilla JS.
- **Dynamic Attachment**: Works on any existing or future `<input type="text">` and `<textarea>` elements globally on the page.
- **Framework Friendly**: Dispatches native JS `input` events, making it fully compatible with **React, Vue, and Angular** data binding.
- **Smart Toggle**: Press `Ctrl + Space` seamlessly to switch between Sinhala and English typing modes.

## Installation / Usage

Simply clone this repository or download `singlish.js`, then include it at the bottom of your HTML document before the closing `</body>` tag:

```html
<script src="path/to/singlish.js"></script>
```
